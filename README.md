# Hercules-GTK3-theme
GTK2/GTK3/Gnome Shell Theme

To use Midnight Commander Skins, copy hercules.ini file to /usr/share/mc/skins/ folder and replece the line skin=default to skin=hercules in ~/.config/mc/ini file
